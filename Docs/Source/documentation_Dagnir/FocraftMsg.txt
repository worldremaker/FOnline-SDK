

FOCRAFT.msg



FOCRAFT.msg - файл, в котором кодируется всё необходимое для работы фикс-боя.



\b Формат:

{Номер строки}{}{PID_ПРЕДМЕТА@Описание.@Требования для просмотра рецепта@Требования для крафта.@Ингридиенты@Инструменты.@Выход предмета.@Экспа, скрипты.}

Используемые операторы:
& = "AND".
| = "OR".

\b Пример:

{27}{}{PID_MOLOTOV_COCKTAIL@Пойло, смешанное со смолянистым соком цветков, будет замечательно гореть и прилипать к противнику.@SK_SCIENCE 20@SK_SCIENCE 30&SK_OUTDOORSMAN 30@PID_ROT_GUT 1&PID_RAGS 1&PID_BROC_FLOWER 2@@PID_MOLOTOV_COCKTAIL 1@exp 50 }

При указании названий прототипов необходимых ингридиентов/инструментов всегда необходимо указывать их количество, даже если оно = 1, иначе это вызовет ошибку сервера при запуске.


Для составления списка рецептов крафта с нуля удобнее всего пользоваться редактором <b>MrFixit</b>.


При выборе номера рецепта желательно оставлять пропуске в списке (скажем, вместо "1, 2, 3" использовать "10, 20, 30", т.к. номера строк должны идти строго по-порядку, и при отсутствии пробелов невозможно будет вставить новые рецепты в середину списка, если возникнет такая необходимость.


В скриптах можно выставлять дополнительные условия для крафта - скажем, проверку на то, находится ли рядом с игроком определённый объект сценери (для примеров см. скрипт fix_boy.fos).
