
\b FODLG.msg



В файле FODLG.msg содержится описание нпц по-умолчанию, т.е. тех, кому не присвоен свой уникальный диалог с описаниями.

<b>Формат:</b>

{PID нпц*10}{}{Имя/название}
{(PID нпц*10)+1}{}{Расширенное описание}

<b>Пример:</b>

{90}{}{Пес}
{91}{}{Собака, наверняка бездомная, дворняжьей породы.}

Такое описание будет у всех нпц с PID=9.



Обычно для присвоения нового описания рекомендуется просто присвоить нпц свой диалог, чтобы не создавать много лишних прототипов с одинаковыми характеристиками. Однако, даже если в файле диалога нет никаких фраз, то во всплывающем меню, появляющемся при нажатии на нпц, всё равно первой опцией будет "начать диалог" (по-умолчанию - бинокль). А это может мешать в бою - при клике по нпц с целью посмотреть параметры персонаж пытается подойти и поговорить с ним. Поэтому, если нужны какие-то нпц без диалогов, которые активно будут использоваться в боевых действиях, лучше создать им новые прототипы со своим описанием для удобства игрока.
