/** \~russian @page page_generateworldcfg GenerateWorld.cfg

GenerateWorld.cfg - список локаций, создаваемых при генерации мира (происходит при первом запуске сервера).

\b Формат:
@code
@ <PID локации> <Координата X> <Координата Y>  # Имя локации
@endcode

\b Примечание:\n
текст, идущий после "#", является комментарием и не влияет на анализ файла.

\b Пример\n
@code
@  13   675   525 # Redding
@endcode
*/
