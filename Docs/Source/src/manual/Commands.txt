/**

\~russian @page commands Базовые команды серверу

@section commands_info Общая информация

Данные команды нужны для облегчения тестирования, отладки игрового процесса. Текущий набор является базовым, встроенным в движок, но с помощью скриптов вы можете существенно расширить функционал до необходимых вам пределов.

Для запуска команды перед ней должен стоять знак "~" (тильда). Так же возможен вызов команды по номеру, для этого наберите два данных знака, а затем номер команды.<br />
После отправки команды на выполнение сервер ответит вам о результате выполнения операции.

@section commands_list Список команд

@par 1. exit<br />
Уровень доступа: любой.<br />
Выход из игры.

@par 2. myinfo<br />
Уровень доступа: любой.<br />
Запрос отображения некоторой информации о текущем персонаже - идентификатор, имя, пароль, уровень доступа.

@par 3. gameinfo \<type\><br />
Уровень доступа: модератор.<br />
Запрос отображения информации об игре, где тип запрашиваемой информации (type) может быть следующим:<br />
- 0) статистика использования памяти;<br />
- 1) сводка по текущим игрокам в игре;<br />
- 2) статистика локаций и карт;<br />
- 3) текущие временные события;<br />
- 4) текущее состояние данных @ref AnyData;<br />
- 5) статистика предметов в игре;

@par 4. id \<name\><br />
Уровень доступа: модератор.<br />
Запрос идентификатора криттера по его имени (name).
@remarks На данный момент не активна.

@par 5. movecrit \<id\> \<hx\> \<hy\><br />
Уровень доступа: модератор.<br />
Передвинуть персонажа (id) в определенный гекс (hx, hy).

@par 6. kill \<id\><br />
Уровень доступа: модератор.<br />
Убить персонажа (id).

@par 7. disconnect \<id\><br />
Уровень доступа: модератор.<br />
Отсоединить персонажа (id) от сервера.

@par 8. toglobal<br />
Уровень доступа: тестер.<br />
Выйти текущим персонажем на глобальную карту.

@par 9. respawn \<id\><br />
Уровень доступа: модератор.<br />
Воскресить персонажа (id). Если id равен 0, то значит воскресить текущего персонажа.

@par 10. param \<type\> \<num\> \<value\><br />
Уровень доступа: тестер.<br />
Изменить параметр у текущего персонажа.<br />
type должен быть равен нулю, num номеру параметра, value его новому значению.

@par 11. getaccess \<access name\> \<access password\><br />
Уровень доступа: любой.<br />
Изменить уровень доступа.<br />
access name - имя доступа (client, tester, moder, admin).<br />
access password - пароль доступа.

@par 12. crash<br />
Уровень доступа: имплементор.<br />
Уронить серверное приложение.

@par 13. additem \<hx\> \<hy\> \<pid\> \<count\><br />
Уровень доступа: модератор.<br />
Добавить предмет (pid, count) на определенный гекс (hx, hy).

@par 14. additemself, ais \<pid\> \<count\><br />
Уровень доступа: модератор.<br />
Добавить предмет (pid, count) в инвентарь.

@par 15. addnpc \<hx\> \<hy\> \<dir\> \<pid\><br />
Уровень доступа: модератор.<br />
Добавить НПЦ (pid, dir) на определенный гекс (hx, hy).

@par 16. addloc \<wx\> \<wy\> \<pid\><br />
Уровень доступа: модератор.<br />
Добавить локацию (pid) в определенную точку (wx, wy) на @ref term_global_map "глобальной карте".

@par 17. reloadscripts<br />
Уровень доступа: администратор.<br />
Перезагрузить все серверные скрипты.

@par 18. loadscript, load \<script name\><br />
Уровень доступа: администратор.<br />
Загрузить или перезагрузить определенный серверный скрипт (script name).

@par 19. reloadclientscripts, rcs<br />
Уровень доступа: администратор.<br />
Перезагрузить клиентские скрипты.

@par 20. runscript, run \<module name\> \<func name\> \<param0\> \<param1\> \<param2\><br />
Уровень доступа: администратор.<br />
Запустить специальную серверную функцию (module name, func name), с определенными параметрами (param0, param1, param2).

@par 21. reloadlocations<br />
Уровень доступа: администратор.<br />
Перезагрузить прототипы локаций.

@par 22. loadlocation \<pid\><br />
Уровень доступа: администратор.<br />
Загрузить определенный прототип (pid) локации.

@par 23. reloadmaps<br />
Уровень доступа: администратор.<br />
Перезагрузить прототипы карт.

@par 24. loadmap \<pid\><br />
Уровень доступа: администратор.<br />
Загрузить определенный прототип (pid) карты.

@par 25. regenmap<br />
Уровень доступа: администратор.<br />
Перегенерировать карту на которой в текущий момент находится персонаж.
@remarks На данный момент работает со сбоями.

@par 26. reloaddialogs<br />
Уровень доступа: администратор.<br />
Перезагрузить все диалоги.

@par 27. loaddialog \<dialog name\> \<dialog id\><br />
Уровень доступа: администратор.<br />
Загрузить определенный диалог (dialog name), под определенным идентификатором (dialog id).

@par 28. reloadtexts<br />
Уровень доступа: администратор.<br />
Перезагрузить все MSG файлы.

@par 29. reloadai<br />
Уровень доступа: администратор.<br />
Перезагрузить сумки НПЦ (Bags.cfg).

@par 30. checkvar, cvar \<tid var\> \<master is npc\> \<master id\> \<slave id\> \<full_info\><br />
Уровень доступа: модератор.<br />
Проверить состояние определенной игровой переменной.<br />
tid var - номер переменной;<br />
master is npc - является ли master id от НПЦ, 0 - нет, остальное - да. Если да, то к master id автоматически добавляется 5kk, т.к. индексация НПЦ начинается с этого числа;<br />
master id - первичный идентификатор;<br />
slave id - вторичный идентификатор;<br />
full_info - выводить ли полную информацию по переменной, 0 - нет, остальное - да.

@par 31. setvar, svar \<tid var\> \<master is npc\> \<master id\> \<slave id\> \<value\><br />
Уровень доступа: модератор.<br />
Установить состояние определенной игровой переменной.<br />
tid var - номер переменной;<br />
master is npc - является ли master id от НПЦ, 0 - нет, остальное - да. Если да, то к master id автоматически добавляется 5kk, т.к. индексация НПЦ начинается с этого числа;<br />
master id - первичный идентификатор;<br />
slave id - вторичный идентификатор;<br />
value - новое значение переменной.

@par 32. settime \<tmul\> \<year\> \<month\> \<day\> \<hour\> \<minute\> \<second\><br />
Уровень доступа: администратор.<br />
Изменить игровое время.<br />
tmul - ускорение времени, 0 - не менять;<br />
year - год, 0 - не менять;<br />
month - месяц, 0 - не менять;<br />
day - день, 0 - не менять;<br />
hour - час;<br />
minute - минута;<br />
second - секунда;<br />

@par 33. ban \<add,add+,delete,list\> \<user\> \<hours\> \<comment\><br />
Уровень доступа: модератор.<br />
Забанить определенного игрока.<br />
add - добавить в бан персонажа;<br />
add+ - добавить в бан персонажа и IP адрес;<br />
delete - удалить из списка забаненных;<br />
list - вывести список забаненных;<br />
user - имя персонажа;<br />
hours - время блокировки, в часах;<br />
comment - комментарий, можно не писать.

@par 34. deleteself \<password\><br />
Уровень доступа: любой.<br />
Удалить текущего персонажа.

@par 35. changepassword, changepass \<password\> \<new password\><br />
Уровень доступа: любой.<br />
Изменить пароль для текущего персонажа.

@par 36. dropuid, drop<br />
Уровень доступа: тестер.<br />
Сбросить антиальт таймер для данного персонажа, что бы можно было перезайти за другого персонажа без ожидания истечения таймаута.

 */
