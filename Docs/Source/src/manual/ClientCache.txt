/** \~russian @page page_clientcache Кэш-файл клиента

В папке Data клиента существует файл dafault.cache, так же, в процессе соединения с сервером, появляются дополнительные файлы с именем в формате:
@code
<ip-адрес>.<порт>.cache
@endcode
Данные файлы являются файлами кэша клиента. В них хранится дополнительная информация о каждом из серверов, с которым было соединение.\n
В файле default.cache содержится информация, загружаемая независимо от того, к какому серверу будет производиться подключение.\n
В данных файлах хранятся такие вещи как: клиентские скрипты, тексты, прототипы предметов, отправленных сервером.\n
Так как интерфейс клиента отрисовывается с помощью скриптов, отсутствие "свежего" файла кэша может приводить к проблемам отрисовки всех элементов интерфейса игры.\n
Для решения проблемы достаточно обновить файл кэша (нажав клавишу Enter клиент попытается подключиться к серверу и, если сервер будет включен, то кэш автоматически обновится).\n
В более новых версиях клиента введена поддержка автоматического обновления файла кэша сразу после запуска.
*/
