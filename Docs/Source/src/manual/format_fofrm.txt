Спецификация .fofrm

@code
fps
@endcode
Frame Per Second, количество кадров в секунду.
Целое, беззнаковое. Не обязательный параметр.
По умолчанию - 10.

@code
count
@endcode
Количество кадров в анимации.
Целое, беззнаковое.
По умолчанию - 1.

@code
offs_x
offs_y
@endcode
Базовые смещения.
Этот сдвиг нужен для того, чтобы кадры были правильно отцентрованы.
Используется при отображении кадра, так как его логический центр не всегда совпадает с его абсолютным центром (width/2, height/2). Например, центром существа будет позиция его ног, в то время как центром дерева будет его основание.
Положительное значение для offs_x означает, что кадр при отображении будет сдвинут вправо.
Положительное значение для offs_y означает, что кадр при отображении будет сдвинут вниз.
Целое, знаковое. Не обязательный параметр.
По умолчанию - 0.

@code
[dir_?]
@endcode
Перед блоками описаний кадров анимации ставится направление анимации.
@b ? - это направление, в диапазоне от 0 до 5.
В кадрах без учета направления это можно не указывать.
Далее следуют элементы блока описаний анимации.

@code
offs_x
offs_y
@endcode
Смещения для направления. Если они не заданы, то берутся глобальные базовые смещения.
Целое, знаковое. Не обязательный параметр.
По умолчанию - значение базовых глобальных offs_*.

@code
next_x_?
next_y_?
@endcode
Смещение кадра относительно предыдущего кадра. ? означает номер кадра*.
Целое, знаковое. Не обязательный параметр.
По умолчанию - 0.

@code
frm_?
@endcode
Имя файла изображения. ? означает номер кадра*.
Для первого кадра номер указывать не обязательно, то есть просто "frm=...".
Должен включать путь от текущей папки.
Строка. Обязательный параметр.

*Номера должны начинаться с 0 и идти по порядку.

*/
