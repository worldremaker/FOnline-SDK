После выгрузки сервера настройте пути к MASTER.DAT и CRITTER.DAT в FOnline.cfg, или просто скопируйте эти два файла в папку клиента.

Сервер, клиент, инструментарий уже синхронизированы между собой.

Структура репозитария:

- Root                           Корневая директория
  SvnClear.bat                   Очистка всего репозитария от рабочих файлов SVN
  License.txt                    Лицензия
  ReadMe.txt                     Данный файл

-- Client                        Клиентское приложение
   d3dx9_42.dll                  Необходимая для работы клиента библиотека Direct3d
   ZipPatch.bat                  Создает патч fonline.zip (или fonlineXXX.zip) автоматически, после чего удаляет контет
   7za.exe                       Архиватор, необходим для работы ClientPatchZip
   FOConfig.exe                  Конфигуратор, работает с файлом FOnline.cfg
   FOnline.exe                   Клиент
   Updater.exe                   Программа для автоматического обновления контента
   NameColorizing.txt            Список имен для работы механизма раскрашивания контуров и ников
   FOnline.cfg                   Конфиг файл клиента
   cablack.dat                   Анимация черной брони
   longhairdude.dat              Анимация чувака с длинными волосами
   vaultboy.dat                  Анимация ВайултБоя из Тактикса
   FOnlineSingleplayer.exe.lnk   Ярлык на запуск однопользовательского режима
--- data                         Данные клиента
    ...

-- Docs                          Документация
   FOnlineRu.chm                 Основная документация по SDK на русском языке

-- Stable                        Стабильные версии SDK, выходят раз в несколько месяцев
   FOnline_vX_DDMMYY.zip         Архив с нужными файлами (внимание: обязательно прочтите файл MissedFiles.txt)

-- Server                        Сервер
   Clean.bat                     Батник для очистки закешированных файлов данных сервера
   FOnlineServer.exe             Серверное приложение
   FOnlineServer.cfg             Конфиг файл сервера
   FOServerService.exe.lnk       Регистрация (запуск, если уже зарегестрирован) сервера как сервис (служба)
   FOServerServiceDelete.exe.lnk Удаление зарегестрированного сервиса (службы)
--- data
    ...
--- dialogs                      Диалоги
    dialogs.lst                  Список диалогов и соответствующих им идентификаторов
    *.fodlg                      Файлы диалогов, открываются с помощью DialogEditor
--- maps                         Карты
    ...
--- proto                        Прототипы
    ...
--- save                         Сохраненные данные о мире и клиентах
    ...

    ...
--- text                         Тексты

    ...
--- DialogEditor                 Редактор диалогов
    ...
--- Mapper                       Маппер
    ...
--- MrFixit                      Редактор крафта
    ...
--- ObjectEditor                 Редактор объектов
    ...
--- Updater                      Апдейтер, сервер и клиент
    ...
