Теги:

@pname@			Имя игрока.
Имя игрока: "player"
Диалог: "Привет @pname@."
Итог: "Привет player."

@nname@			Имя Нпц.
Имя игрока: "npc"
Диалог: "Привет @nname@."
Итог: "Привет npc."

@@			Разделяет фразы, которые в последствии будут выбраны рандомным образом.
Диалог: "Привет.@@Привет!@@Здаров."
Итог: "Привет." или "Привет!" или "Здаров."

@sex@			После него, если пол игрока мужской, то появляется текст из первого отрезка разделенного '|'. Второй отрезок пропускается. Если женский, то наоборот.
Пол игрока: женский
Диалог: "Привет @sex@|мужик||женщина|."
Итог: "Привет женщина."

@msg num str@		Вставка строки из MSG, где num номер MSG файла, а str номер строки.
Диалог: "Привет@msg 0 100@"
Стока в FOTEXT.MSG: "{100}{}{"!"}"
Итог: "Привет!"

@lex name@		Вставка строки передаваемой в дополнительной строке. name - имя лексемы.
Диалог: "Для вас это будет стоить @lex money@ монет. А это @lex money_2@ монет."
Передаваемая строка лексем: "$money1000$money_2500"
Итог: "Для вас это будет стоить 1000 монет. А это 500 монет."

@script func_name@	Вставка строки возвращаемой функцией в скрипте клиента.
Диалог: "Для вас это будет стоить @script money@ монет."
Скрипт: string _Func(string& lexems){return "999";}
Итог: "Для вас это будет стоить 999 монет."
********************************************************************
