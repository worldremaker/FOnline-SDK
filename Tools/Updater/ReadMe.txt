Для корректной работы системы сделайте следующее:
0. Опционально - сообщите администрации проекта о включении вашего сервера в список (irc: forestnet.org #fonline);
1. В папку Server\Files скопируйте все файлы необходимые игроку для запуска игры;
2. Запустите Server\UpdaterServer.exe, подождите пока файлы будут проиндексированы, нажмите Start listening;
3. Распространите файл Client\Launcher.exe между игроками, он будет использован для установки и обновлений игры.
